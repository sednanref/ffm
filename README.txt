INSTRUCTIONS
============

To run the programm you just need to:
	* have Python installed. 
	* Type in the command line: python ffm.py
	* Follow the instructions provided by the programm.
	* Have fun!

On each iteration you will see a grid composed by the characters:
	'space': denoting a cell with Ashes.
        'T': denoting a cell with Trees.
        '*': denoting a cell with Trees burning.

On each iteration you will find the number of cells with ahses, cells with trees and cells with burning trees.
